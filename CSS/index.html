Introduction

JavaScript is a versatile scripting language used mainly for web development.

It allows you to add interactivity to websites, handle events, and manipulate the DOM.

JavaScript runs in the browser and on servers using Node.js.

console.log("Hello, world!");

You can embed JavaScript directly in HTML or load it from external files.

  • Client-side scripting
  • Lightweight and fast
  • Interpreted language
Variables

Variables are containers for storing data values.

You can declare them using var, let, or const.

let name = "Alice";

`let` and `const` were introduced in ES6.

const PI = 3.14;
Functions

Functions are blocks of code designed to perform a task.

They can take parameters and return values.

function add(a, b) { return a + b; }

Functions help you reuse code.

  • Named functions
  • Arrow functions
  • Callback functions
Loops

Loops allow you to repeat blocks of code.

Types of loops: for, while, do...while.

for (let i = 0; i < 5; i++) { console.log(i); }

Be careful of infinite loops.

while (condition) { /* code */ }
Arrays

Arrays are used to store multiple values in a single variable.

They are zero-indexed and can hold different data types.

let fruits = ["apple", "banana", "cherry"];

Common methods include push(), pop(), and map().

  • Indexed collection
  • Mutable
  • Length property
